INSTALLATION

RECOMMENDED

One-Click Installation

Download the MCP Bundle for instant installation. Works with Claude Desktop, Cursor, Windsurf, and other MCP-compatible clients.

📦 DOWNLOAD CREATIVE-JUICES.MCPB
After downloading: Double-click the file to install. Your MCP client will handle the rest. No configuration needed.

⚙️ Advanced: Manual Installation

Prefer to configure manually? Expand your client below for detailed instructions.

Claude Desktop

Edit: ~/Library/Application Support/Claude/claude_desktop_config.json 

{
  "mcpServers": {
    "creative-juices": {
      "url": "https://ai.yuda.me/mcp/creative-juices/serve"
    }
  }
}

After saving, restart Claude Desktop (Cmd+Q then reopen). Look for the 🔌 icon.

Claude Code (CLI)

Run this command in your terminal: 

claude mcp add creative-juices --url https://ai.yuda.me/mcp/creative-juices/serve

Installation

1. Clone the Repository

git clone https://github.com/tomcounsell/cuttlefish.git
cd cuttlefish
uv sync --all-extras

2. Test the Server

# Run with MCP Inspector
npx @modelcontextprotocol/inspector uv run python -m apps.ai.mcp.creative_juices_server

3. Configure Claude Desktop

Add to your claude_desktop_config.json:

{
  "mcpServers": {
    "creative-juices": {
      "command": "uv",
      "args": [
        "--directory",
        "/ABSOLUTE/PATH/TO/cuttlefish",
        "run",
        "python",
        "-m",
        "apps.ai.mcp.creative_juices_server"
      ]
    }
  }
}

Config file location:

  • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows: %APPDATA%\Claude\claude_desktop_config.json

Replace /ABSOLUTE/PATH/TO/cuttlefish with your actual repository path, then restart Claude Desktop.

Why Use Creative Juices?

  • Break habitual thinking - Random concrete metaphors force new mental pathways
  • Zero setup - No API keys, no configuration, works immediately
  • Completely private - Runs locally, no external calls, no data collection
  • Battle-tested frameworks - Strategic questions from proven problem-solving patterns
  • 600+ curated words - Spanning human history from primitive tools to sci-fi futures

Design Philosophy

Concrete over Abstract: Words like "baking-shoe" create stronger creative friction than "crystallize-entropy" because the conceptual gap is larger.

Three-stage process: Diverge gently (inspiring) → Diverge dramatically (out-of-the-box) → Converge strategically (reality check)

Historical dimension: Words span from primitive era (knapping, flint) through ancient civilizations (forging, wheel) to futuristic sci-fi (teleporting, quantum-drive) for maximum metaphorical range.

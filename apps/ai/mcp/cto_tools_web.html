🎯 CTO Tools

Engineering leadership frameworks for AI assistants

What is CTO Tools?

CTO Tools is a Model Context Protocol (MCP) server that provides structured frameworks and best practices for common CTO activities. It helps AI assistants guide engineering leaders through weekly reviews, team analysis, and strategic decision-making.

Available Tools

📊 weekly_review()

Streamlined 3-phase framework for weekly engineering team reviews. Analysis-focused, not prescriptive. Works with ANY codebase.

  • Phase 1: Data gathering with parallel git commands
  • Phase 2: Internal analysis (review → categorize → stats)
  • Phase 3: Concise output (~200 words) with 5 categories and contributor stats

Features: ⚡ Fast reviews (15-20 min) | 📊 Concise output | 👥 Automatic contributor recognition | 🎯 Communication-ready format

Features

📈 3-Phase Framework

Streamlined process: gather data, analyze internally, output concise summary

🔍 Adaptive Categorization

AI chooses 5 relevant categories based on actual work, not templates

📝 Concise Output

Single ~200 word summary with bullets, suitable for any communication channel

🎯 Focus on What Changed

Clear deliverables without verbose analysis - thinking happens internally

Installation - Single Executable File

No dependencies required! Run directly from URL with uv:

{
  "mcpServers": {
    "cto-tools": {
      "command": "uv",
      "args": [
        "run",
        "https://raw.githubusercontent.com/tomcounsell/cuttlefish/main/apps/ai/mcp/cto_tools_server.py"
      ]
    }
  }
}

Or download and run locally: 

curl -O https://raw.githubusercontent.com/tomcounsell/cuttlefish/main/apps/ai/mcp/cto_tools_server.py
chmod +x cto_tools_server.py
./cto_tools_server.py

Web Manifest: https://ai.yuda.me/mcp/cto-tools/manifest.json

How It Works

  1. Request a Tool: Ask Claude to use CTO Tools for team review
  2. Get Framework: Receive structured instructions and best practices
  3. Follow Steps: Use the provided git commands and analysis frameworks
  4. Generate Insights: Create executive summaries and action items

CTO TOOLS

ENGINEERING LEADERSHIP FRAMEWORKS

CTO Tools provides structured frameworks and best practices for common engineering leadership activities. Helps AI assistants guide technical leaders through weekly reviews, team analysis, and strategic decision-making with analysis-focused, non-prescriptive approaches.

AVAILABLE TOOLS

TOOL_01

weekly_review(days=7, categories=5)

Streamlined 3-phase framework for engineering team reviews with flexible time periods. Analysis-focused, not prescriptive. Works with ANY codebase.

Parameters

  • days (int, default: 7): Number of days to review (e.g., 7 for weekly, 14 for bi-weekly, 30 for monthly)
  • categories (int, default: 5): Number of work categories to organize the output into

Phase 1: Data Gathering

  • Parallel git commands for efficient data collection
  • Commit analysis across the reporting period (automatically adjusted based on days parameter)
  • Contributor activity tracking

Phase 2: Internal Analysis

  • Review commits and categorize work
  • Identify patterns and themes
  • Generate contributor statistics

Phase 3: Concise Output

  • Plain text (.txt) format with full Unicode emoji support
  • Concise summary with N adaptive categories (based on categories parameter)
  • Contributor recognition and stats
  • Communication-ready format for any channel - copy-paste anywhere
Characteristic Implementation
Review Time 15-20 minutes
Time Period Flexible (default: 7 days)
Categories Flexible (default: 5), based on actual work
Output Format Plain text (.txt) with Unicode emoji support
Codebase Universal, no configuration needed

INSTALLATION

01

curl -LsSf https://astral.sh/uv/install.sh | sh
02

Add to Claude Desktop config

Edit: ~/Library/Application Support/Claude/claude_desktop_config.json 

{
  "mcpServers": {
    "cto-tools": {
      "command": "uvx",
      "args": [
        "run",
        "https://raw.githubusercontent.com/tomcounsell/cuttlefish/main/apps/ai/mcp/cto_tools_server.py"
      ]
    }
  }
}
03

Restart Claude Desktop

Completely quit Claude Desktop (Cmd+Q on macOS) and reopen. Look for the 🔌 icon in the bottom-right corner.

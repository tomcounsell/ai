Design Rules

Border Radius Policy

No border radius
Containers, cards, sections, divs, code blocks
Allowed border radius
Buttons (subtle default rounding), status indicators (50% for circles)

Typography Rules

Headings (h1-h3) Sans-serif (Inter), 600 weight, uppercase, 0.05em tracking
Labels Monospace, uppercase, 0.75rem, 0.08em letter-spacing
Body text Sans-serif, normal case, 0.875rem, 1.8 line-height
Code and data Monospace, normal case, 0.75-0.875rem
Buttons Monospace, uppercase, 0.75rem, 0.1em letter-spacing

Spacing Guidelines

Between sections --space-2xl (64px) or --space-3xl (96px)
Between components --space-lg (32px) or --space-xl (48px)
Within components --space-sm (16px) or --space-md (24px)
Micro-spacing --space-xs (8px)

Label Naming Convention

Technical documentation pattern with underscores and zero-padded numbers for data labels and identifiers:

MCP_SERVER_01
TOOL_03
BENEFIT_02
INSTALLATION_GUIDE
PROTOCOL_OVERVIEW

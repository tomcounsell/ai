03_SPACING_SYSTEM

8px Base Grid

All spacing values are multiples of 8px for mathematical harmony and consistent rhythm.

--space-xs 8px Micro-spacing, tight gaps
--space-sm 16px Within components
--space-md 24px Component spacing
--space-lg 32px Between components
--space-xl 48px Large gaps
--space-2xl 64px Section padding
--space-3xl 96px Between sections

01

Install uv (if needed)

Add to Claude Desktop config

Edit: ~/Library/Application Support/Claude/claude_desktop_config.json 

{
  "mcpServers": {
    "server-name": {
      "command": "uvx",
      "args": ["run", "https://example.com/server.py"]
    }
  }
}
03

Restart Claude Desktop

Completely quit Claude Desktop (Cmd+Q on macOS) and reopen. Look for the 🔌 icon in the bottom-right corner.

05_USAGE_RULES

Border Radius Policy

NO_BORDER_RADIUS
Containers, cards, sections, divs, code blocks
ALLOWED_BORDER_RADIUS
Buttons (subtle default rounding), status indicators (50% for circles)

Typography Rules

LABELS Monospace, uppercase, 0.75rem, 0.08em letter-spacing
HEADERS Sans-serif or monospace, uppercase, increased letter-spacing
BODY_TEXT Sans-serif, normal case, 0.875-0.95rem
CODE_DATA Monospace, normal case, 0.75-0.875rem
BUTTONS Monospace, uppercase, 0.688rem, 0.1em letter-spacing

Spacing Guidelines

BETWEEN_SECTIONS --space-2xl (64px) or --space-3xl (96px)
BETWEEN_COMPONENTS --space-lg (32px) or --space-xl (48px)
WITHIN_COMPONENTS --space-sm (16px) or --space-md (24px)
MICRO_SPACING --space-xs (8px)

Label Naming Convention

Follow technical documentation pattern with underscores and zero-padded numbers:

MCP_SERVER_01
TOOL_03
BENEFIT_02
INSTALLATION_GUIDE
PROTOCOL_OVERVIEW

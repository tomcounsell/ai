What is MCP?

Model Context Protocol - The standard for AI assistant integrations

MCP (Model Context Protocol) is an open standard that enables AI assistants like Claude to securely connect with external tools and data sources. Think of it as a universal plugin system for AI.

Quick Start

Get up and running in under 5 minutes

1

Install uv (if needed)

curl -LsSf https://astral.sh/uv/install.sh | sh
2

Add to Claude Desktop config

Edit: ~/Library/Application Support/Claude/claude_desktop_config.json 

{
  "mcpServers": {
    "creative-juices": {
      "command": "uvx",
      "args": [
        "run",
        "https://raw.githubusercontent.com/tomcounsell/cuttlefish/main/apps/ai/mcp/creative_juices_server.py"
      ]
    }
  }
}
3

Restart Claude Desktop

Completely quit Claude Desktop (Cmd+Q on macOS) and reopen. Look for the 🔌 icon in the bottom-right corner.
